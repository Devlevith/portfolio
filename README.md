# portfolio

--- Sobre o Projeto ---
Este é meu portfólio pessoal e centralizador de links, desenvolvido para apresentar minha jornada como Criador Digital e Desenvolvedor em formação. O objetivo foi criar uma interface visualmente atraente que conectasse minhas diferentes facetas: desenvolvimento de jogos, criação de conteúdo e design.

--- Teoria e Construção (Como foi feito) ---
O projeto foi construído com foco em simplicidade técnica e impacto visual, utilizando o conceito de "Mobile First" adaptado para uma experiência de Grid em Desktop.

Arquitetura: Desenvolvido puramente com HTML5 e CSS3, sem dependências de frameworks pesados, garantindo um carregamento instantâneo.

Design: Utilizei variáveis CSS (:root) para manter a consistência das cores (tons de roxo e escuros) e efeitos de backdrop-filter: blur para criar a profundidade nos cards.

Dinamismo: Implementei um sistema simples de Feed via JavaScript, onde os itens são injetados dinamicamente no DOM, facilitando futuras atualizações sem mexer na estrutura HTML.

Engajamento: Criei um sistema de "Anúncio Fixo" (Fixed Ad) programado em JavaScript que utiliza setInterval e manipulação de classes CSS para aparecer e desaparecer em intervalos específicos, simulando uma interação de marketing para o coletivo Middle Creative.
